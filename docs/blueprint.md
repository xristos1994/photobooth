# **App Name**: PicClick Booth

## Core Features:

- Photo Capture: Capture photos using the user's webcam with countdown timers (5 seconds for the first, 3 for subsequent shots).
- Layout Selection: Allow the user to select the number of photos to take (1-4).
- Live Preview: Display a live preview of the webcam feed.
- Photo Display: Display the captured photos in a vertical stack to the right of the preview.
- Image Merging: Merge the captured photos into a single image.
- Automatic Download: Automatically download the merged image as a single file.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to evoke the excitement and fun of a photo booth.
- Background color: Light grayish-blue (#E0F7FA) to keep the focus on the photos.
- Accent color: Soft pink (#FF69B4) for a touch of playfulness in buttons and highlights.
- Headline font: 'Poppins' sans-serif for a clean and modern feel.
- Body font: 'PT Sans' sans-serif, used when there is on-screen text, or help content.
- Left side: Live preview, controls for photo count, and the 'Start' button. Right side: Vertically stacked captured photos.
- Subtle transitions when capturing and merging photos.
